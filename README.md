## Project Overview

AudioBooks App is a cross-platform mobile application built using React Native.
It provides users with a seamless experience to:

Browse available audiobooks

View book details

Download book in pdf form

Play audio with controls (play/pause/seek)

Track listening progress

Manage user profile

This project focuses on mobile UI/UX, state management, and API integration.
